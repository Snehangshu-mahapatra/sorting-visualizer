# Sorting Algorithm Visualizer

An interactive web application that visualizes sorting algorithms in real time.

## Features
- Bubble Sort and Selection Sort visualization
- Adjustable speed control
- Performance tracking (comparisons and swaps)
- Dynamic bar animations

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in a browser or use Live Server in VS Code.

## Future Improvements
- Add more algorithms (Insertion, Merge, Quick Sort)
- Add side-by-side comparison
